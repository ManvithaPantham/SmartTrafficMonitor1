# 🚦 Smart Traffic Monitoring System using AI

## 📌 Overview

An AI-powered Smart Traffic Monitoring System that detects, classifies, and counts vehicles from traffic videos using **YOLOv8** and **OpenCV**. The project features a **Flask** backend, **MongoDB** for data storage, and an interactive dashboard for real-time traffic analytics.

## ✨ Features

* 🚗 Real-time vehicle detection and counting
* 🚙 Vehicle classification (Car, Bike, Bus, Truck)
* 📊 Interactive traffic analytics dashboard
* 🎥 Live traffic camera streaming
* 🚦 Dynamic traffic signal indicator
* 🗄️ MongoDB database integration
* 📈 Real-time data visualization with Chart.js

## 🛠️ Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* Flask
* MongoDB
* HTML, CSS, JavaScript
* Chart.js

## 🚀 How to Run

1. Install the required Python libraries.
2. Run `detection.py`.
3. Run `server.py`.
4. Run `camera_server.py`.
5. Open `dashboard.html` in your browser.

## 📸 Output

* Vehicle detection with labels and IDs
* Real-time vehicle count
* Live traffic camera feed
* Interactive dashboard with analytics and traffic status
